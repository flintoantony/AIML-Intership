# 🚗 CO2 Emission Data Analysis & Machine Learning Project

## 📌 Overview

This project focuses on analyzing a CO2 Emission dataset using **Exploratory Data Analysis (EDA)** techniques and basic **Machine Learning concepts**.
The objective is to understand the factors influencing vehicle carbon emissions and extract meaningful insights through data visualization.

## 📊 Exploratory Data Analysis (EDA)

EDA was performed to analyze patterns and relationships in the dataset.

### Visualizations Included:

* 📈 Histograms
* 🔥 Correlation Heatmap
* 📊 Scatter Plots
* 📦 Box Plots

---

## 💡 Key Insights

* Larger engine size vehicles produce higher CO2 emissions
* Vehicles with more cylinders emit more carbon dioxide
* SUVs and heavy vehicles show higher emission levels
* Fuel consumption directly impacts CO2 emissions
* Dataset contains no missing values
